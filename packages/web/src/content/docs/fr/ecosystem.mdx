Une collection de projets communautaires construits sur OpenHei.

:::note
Vous souhaitez ajouter votre projet lié à OpenHei à cette liste ? Soumettez un PR.
:::

Vous pouvez également consulter [awesome-openhei](https://github.com/awesome-openhei/awesome-openhei) et [openhei.cafe](https://openhei.cafe), une communauté qui regroupe l'écosystème OpenHei.
