Eine Sammlung von Community-Projekten, die auf OpenHei basieren.

:::note
Möchten Sie Ihr OpenHei-bezogenes Projekt zu dieser Liste hinzufügen? Senden Sie einen PR.
:::

Sie können sich auch [awesome-openhei](https://github.com/awesome-openhei/awesome-openhei) und [openhei.cafe](https://openhei.cafe) ansehen, eine Community, die das Ökosystem und die Community zusammenfasst.
