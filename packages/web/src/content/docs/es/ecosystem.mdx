Una colección de proyectos comunitarios construidos en OpenHei.

:::note
¿Quieres agregar tu proyecto relacionado con OpenHei a esta lista? Envíe un PR.
:::

También puedes consultar [awesome-openhei](https://github.com/awesome-openhei/awesome-openhei) y [openhei.cafe](https://openhei.cafe), una comunidad que agrega el ecosistema y la comunidad.
