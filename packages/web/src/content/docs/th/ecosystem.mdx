ชุดโครงการชุมชนที่สร้างขึ้นบน OpenHei

:::note
ต้องการเพิ่มโครงการที่เกี่ยวข้องกับ OpenHei ของคุณลงในรายการนี้หรือไม่ ส่งพีอาร์.
:::

คุณยังสามารถลองดู [awesome-openhei](https://github.com/awesome-openhei/awesome-openhei) และ [openhei.cafe](https://openhei.cafe) ซึ่งเป็นชุมชนที่รวบรวมระบบนิเวศและชุมชนไว้ด้วยกัน
